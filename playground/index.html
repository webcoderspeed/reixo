Powerful Features Built In

Result<T,E> Returns

No try/catch hell. Type-safe error handling that makes your code predictable and maintainable.

Smart Retry & Circuit Breaker

Exponential backoff, configurable thresholds, and half-open states. Built for real-world reliability.

Request Deduplication

Identical concurrent requests are automatically merged. Save bandwidth and reduce latency.

OpenTelemetry Tracing

Zero-dependency W3C trace context propagation. Observe every request end-to-end.

GraphQL + WebSocket + SSE

All protocols in one unified API. REST, GraphQL, real-time subscriptions, and more.

Offline Queue

Requests queue automatically when offline. Replay seamlessly when connectivity restores.

Built for Real Development

Clean, intuitive API that feels natural to TypeScript developers

import { reixo } from 'reixo';

const client = reixo.create({
  baseURL: 'https://api.example.com'
});

const result = await client.get<User>('/users/1');

if (result.ok) {
  console.log(result.data); // User
} else {
  console.error(result.error.code); // typed error
}
const client = reixo.create({
  baseURL: 'https://api.example.com',
  retry: {
    attempts: 3,
    delay: 'exponential',
    on: [429, 503],
  },
  circuitBreaker: {
    threshold: 5,
    timeout: 30_000,
  },
});

// Requests automatically retry on 429/503
const result = await client.post('/data', { /* ... */ });
const { data } = await client.graphql<{
  user: User
}>({
  query: `query GetUser($id: ID!) {
    user(id: $id) {
      id
      name
      email
    }
  }`,
  variables: { id: '1' },
});

console.log(data.user.name);
const { data, loading, error } = useReixoQuery<User>(
  '/users/1'
);

if (loading) return <Spinner />;
if (error)   return <ErrorBanner message={error.message} />;
return <UserCard user={data} />;
